# java-cli-gradle-concurrent-hello-world

## Description
A POC for a java concurrent threads.
Threads are `join` to spell 'Hello World'

## Tech stack
- java
- gradle

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
